# Todo App Backend

This is a Todo backend project built using Node.js, Express, and MongoDB.

## Features
- Create Todo
- Update Todo
- Delete Todo
- Get All Todos

## API Endpoints

POST /api/v1/createTodo  
GET /api/v1/getTodos  
PUT /api/v1/updateTodo/:id  
DELETE /api/v1/deleteTodo/:id
